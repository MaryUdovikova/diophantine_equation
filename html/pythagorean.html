Приклад розв'язання

Нехай маємо гіпотенузу c = 5. Потрібно знайти цілі додатні катети a та b, для яких a2 + b2 = c2.

Виразимо c як суму двох квадратів c = m2 + n2.
Маємо 5 = 22 + 12, отже m = 2 та n = 1.
Катети обчислюємо за формулами a = m2 − n2 та b = 2mn. Підставивши, отримуємо a = 22 − 12 = 3 та b = 2×2×1 = 4. Отже, шукані катети - це a = 3 та b = 4.

Перевіримо отримане рішення:
32 + 42 = 52
9 + 16 = 25
25 = 25. Отже, катети знайдені вірно.
